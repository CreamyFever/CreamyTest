한 달 후에 개최될 예정인 캡스톤 경진대회에 대비해서 유니티로 작업 중인 내용의 일부입니다.
즉, 현재진행형으로 작업 및 수정 중입니다.

고전게임 소코반을 벤치마킹한 게임을 제작 중이며, 경진대회 일까지 추가 업데이트 예정입니다.


LevelData - 레벨을 구성하는 데이터가 담긴 클래스입니다.
JsonFileManager - JSON 파일을 읽어들여서 레벨데이터의 리스트에 대입합니다.
GameManager - 앞으로 게임의 상태를 관리할 매니저 클래스. 여기에는 게임 초기화와 레벨 넘어가는 기능 뿐입니다.
StageManager - JsonFileManager로부터 읽어들인 레벨데이터를 씬에 배치합니다.
Cell - 맵 블록, 움직이는 객체들의 상위 클래스입니다.
MovingObject - 게임 상의 모든 움직이는 물체의 상위 클래스입니다.
Player - MovingObject를 상속받은 하위 클래스입니다.
StageEnable - 스테이지 선택 씬의 버튼들을 비활성화 상태로 초기화하고, 클리어한 스테이지 수만큼 버튼을 활성화합니다.
